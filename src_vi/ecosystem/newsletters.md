# Newsletter Cộng đồng {#community-newsletters}

Có nhiều newsletter / blog chuyên về Vue từ cộng đồng cung cấp tin tức và sự kiện mới nhất trong hệ sinh thái Vue. Dưới đây là danh sách các newsletter đang hoạt động mà chúng tôi đã tổng hợp (không đầy đủ):

- [Vue.js Feed](https://vuejsfeed.com/)
- [Michael Thiessen](https://michaelnthiessen.com/newsletter)
- [Jakub Andrzejewski](https://dev.to/jacobandrewsky)
- [Weekly Vue News](https://weekly-vue.news/)
- [Vue.js Developers Newsletter](https://vuejsdevelopers.com/newsletter/)

Nếu bạn biết một newsletter hay chưa được liệt kê, vui lòng gửi pull request qua liên kết bên dưới!
